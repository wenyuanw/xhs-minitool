# 绿茵门神

30 秒冲刺接球小游戏：拖动球门接住足球，漏接扣生命，小心红黄牌。

## 玩法

- **足球 / 花式球 / 金球**：接到得分（连击有加成）
- **生命**：漏接足球扣 1 命，3 命耗尽即结束
- **连击热潮**：连击 5/10/15 触发，球门变宽、掉落略慢
- **必扑球**：高亮金球，接到高分（偶带回血），漏接扣命
- **飞扑**：每局限 2 次，短暂扩大接球范围
- **加时**：+2 秒（有上限）
- **激励**：短时间得分 ×2
- **黄牌**：掉落变慢一段时间
- **红牌**：扣分

难度会随时间上升：掉落更快、刷得更密、球门略缩小、红黄牌更多。
