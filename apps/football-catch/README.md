# 绿茵门神

接住足球，当好门神。

一款 30 秒冲刺的接球小游戏：左右拖动球门，接住从天而降的足球与道具，守住生命，冲刺高分。

---

## 一句话介绍

接住足球，当好门神。

## 游戏简介

你是绿茵场上的门神。比赛开始后，足球、金球、加时、激励以及红黄牌会不断从上方掉落。左右拖动底部球门去接住它们——接球得分、连击升温，漏接则扣命。

每局 **30 秒**、**3 点生命**。时间耗尽或生命归零即结算本场成绩，并记录最高分。越往后掉落越快、刷得更密，球门也会略微缩小，考验手速与判断。

## 基础操作

- 左右拖动底部球门，接住掉落物
- 漏接足球 / 金球 / 必扑球会扣除 1 点生命
- 生命归零或倒计时结束，本场结算
- 建议开启音效，氛围更带感

## 得分与增益

| 道具 | 效果 |
| --- | --- |
| 足球 / 花式球 | 接到得分，连续接球可叠连击加成 |
| 金球 | 高分球，漏接同样扣命 |
| 加时 | 倒计时 +2 秒（有上限） |
| 激励 | 短时间得分 ×2 |

## 进阶机制

| 机制 | 说明 |
| --- | --- |
| 连击热潮 | 连击达到 5 / 10 / 15 时触发：球门变宽，掉落略慢 |
| 必扑球 | 带光圈的高亮球，每局最多 3 个；接到高分（偶带回血），漏接扣命 |
| 飞扑 | 每局限用 2 次，短暂扩大接球范围，关键时刻救场 |

## 惩罚道具

| 道具 | 效果 |
| --- | --- |
| 黄牌 | 掉落变慢一段时间，连击清空 |
| 红牌 | 直接扣分，尽量躲开 |

## 高分小贴士

1. **保住连击**：连击热潮能让局面更从容，是冲分关键。
2. **看好必扑球**：分值高，偶带回血，但漏接必扣命。
3. **飞扑留给危急时刻**：每局只有 2 次，别浪费在轻松球上。
4. **红牌能躲就躲**：黄牌虽会清连击，但能换来一段减速喘息。
5. **后期更密更快**：越往后越要提前预判落点，别只盯着屏幕下方。

## 适合谁玩

适合想在碎片时间里快速开一局、比拼手速与反应的玩家。操作简单，节奏紧凑，适合反复挑战最高纪录。
